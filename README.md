# DI
This package provides easy and convenient management of your dependencies.

## Requirements
| master | 1.0 |
|:----------|:---------------------:|
| php       | \>\=7.2               |

## Installation
Install the latest version with:
```
$ composer require yakiv-khorunzhyi/di-package
```
